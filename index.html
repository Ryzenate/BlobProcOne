              Deployment Setup
            

            

              

                
1

                

                  
Deploy Static Web App

                  
Place index.html and staticwebapp.config.json in your repo root. Connect to GitHub and deploy via Azure Static Web Apps.

                

              

              

                
2

                

                  
Deploy Azure Function API

                  
Place api/process_trigger/__init__.py and function.json in the api/ folder. The SWA runtime auto-links them under /api/.

                

              

              

                
3

                

                  
Configure Application Settings

                  
In the Azure Portal → your Static Web App → Configuration, add:
AZURE_STORAGE_CONNECTION_STRING
BLOB_CONTAINER_NAME

                

              

              

                
4

                

                  
Local CLI Usage

                  
Install deps, then run processor.py directly for offline or bulk work:

                  
pip install -r requirements.txt

python processor.py \
  --zip "C:\Downloads\ASUS_BIOS.zip" \
  --container-path "drivers/ASUS/BIOS/ASUS_BIOS.zip" \
  --connection-string "DefaultEndpoints..." \
  --container-name "updates" \
  --work-dir "C:\Temp\Pipeline"

                

              

            

          


          

            

              
              File Layout
            

            

              
your-repo/
├── index.html               ← this file
├── staticwebapp.config.json ← routing + auth
├── requirements.txt         ← pip deps
├── processor.py             ← pipeline engine
└── api/
    └── process_trigger/
        ├── __init__.py      ← Azure Function
        └── function.json    ← binding config

Azure Blob Container (e.g. "updates"):
├── drivers/
│   ├── ASUS/
│   │   ├── BIOS/
│   │   │   ├── asus_bios_v310.zip     ← source
│   │   │   └── out/
│   │   │       ├── asus_bios_v310_processed.zip
│   │   │       ├── metadirs.json
│   │   │       └── zipmanifest.json
│   │   └── WLAN/
│   └── HP/
│       └── Chipset/
